### 입력 값
<hr />

- [X] 입력 값이 null이거나 빈 공백 문자일 경우 IllegalArgumentException throw
- [X] 사칙 연산이 불가능한 입력 값인 경우 IllegalArgumentException throw
- [X] 정상 입력 값 테스트
- [X] 입력 값으로 들어온 사칙연산 기호를 순서대로 저장하는지 테스트
- [X] 입력 값으로 들어온 숫자를 순서대로 저장하는지 테스트

<br />
